# Pi-Star ⭐

> **Synthesized agent harness.** Fork of [Pi](https://github.com/badlogic/pi-mono) — the minimal transparent terminal coding agent — with added layers for multi-model routing, lean LSP verification, phase-gate governance, methodology feedback loops, and cost-aware model selection.

---

## Status

🚧 **Design phase.** See [ARCHITECTURE.md](ARCHITECTURE.md) for the comprehensive research synthesis and implementation plan.

## Philosophy

- **Token efficiency is the primary design constraint.** Every feature must justify its cost.
- **The user controls the context, not the tool.** Full transparency, no stealth injection.
- **Every decision is evidence-backed.** Features without research ship as optional extensions.
- **Methodology is infrastructure.** Phase gates, verification, and feedback are harness primitives, not scripts.
